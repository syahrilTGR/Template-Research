description: Use this when the user asks 'where should I cite this paper?'. Verify a newly uploaded paper and suggest where to use it in the thesis.
---

1. Read the abstract and introduction of the uploaded or specified paper file.
2. Confirm and report: title, author(s), year, and publication venue.
3. **Verify Valid Range**: Paper should be within 2022–2027 (seminal papers exempt).
4. Determine relevance to project topic and technical stack.
5. Suggest which subsection(s) in Chapter 2 or Chapter 3 can use this paper.
6. Read `supportFiles/pending_references.md` and assign/verify a temporary ID (REF-X).
7. Suggest the correct final reference number based on the current reference map in your handoff files.
8. **Citation Guard**: Never fabricate authors or years.
